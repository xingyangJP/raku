# XERO PM 連携強化アイデア

## 概要
XERO PM（顧客＆日報管理システム）と販売管理のデータを行き来させることで、**案件ライフサイクルを一気通貫で可視化**し、営業～実行～改善までの PDCA を高速化する。すべての仕様は公開 API ドキュメント（<https://api.xerographix.co.jp/public/docs/>）を参照できる。

## 連携戦略
1. **案件／見積トラッキング**
   - 見積承認時に PM の「案件」または「活動」に自動登録し、進捗とステータスを同期。
   - 案件番号やタスク進行率を `/quotes` に表示し、営業が PM の実行状況を把握できるようにする。
2. **日報（実績時間）×見積工数の照合**
   - PM の日報 API から受注案件の実績時間を取得し、工数明細（社内用）に差異を表示。  
   - 乖離が一定値を超えると Slack / Chat 通知を送るなど、原価管理を自動化。
3. **戦略レポート**
   - PM の顧客評価や案件カテゴリ × 見積粗利をレポート化し、取引先別の利益・受注率を分析。
   - 「次に提案すべき顧客」「放置中案件」など営業アクションリストをダッシュボードに表示。
4. **AI 補助見積**
   - 見積書に「要件概要」欄を追加し、PM の顧客情報や日報を元に要約を生成。
   - 要件概要をプロンプトとして OpenAI API を呼び出し、明細案や備考を自動生成する導線を用意する。
   - AI が作成したドラフトを承認フローへ直接流せるようにし、営業の作業時間を削減。

## 期待効果
- 案件・顧客データの重複入力を撤廃し、営業と開発の情報連携を高速化。
- 実績工数と見積工数のギャップを自動で検知し、粗利悪化を早期に発見。
- 顧客行動データ（日報・タスク）と売上データを統合することで、案件ポートフォリオの最適化が可能になる。
